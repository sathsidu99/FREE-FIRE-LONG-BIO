# FreeFire Bio Creator - Vercel Hosted

## Features
- Update FreeFire profile bio
- Multiple login methods (JWT, UID/Pass, Access Token)
- Modern web interface
- Hosted on Vercel

## Setup Instructions

### Deploy to Vercel:
1. Install Vercel CLI: `npm i -g vercel`
2. Login: `vercel login`
3. Deploy: `vercel`

### Or use GitHub:
1. Push to GitHub repository
2. Go to vercel.com
3. Import your GitHub repository
4. Deploy automatically

## API Endpoint
`/api/bio_upload?bio=YOUR_BIO&jwt=YOUR_TOKEN`

## Note
This is a simplified Vercel version. The protobuf functionality is limited.
For full features, use the local Flask version.
